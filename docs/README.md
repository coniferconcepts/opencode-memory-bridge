## Key Concepts

### The Five Layers

1. **Universal Base** - Core agents, patterns, guardrails, workflows
2. **Templates** - Stack-specific project templates
3. **Project Layer** - Project-specific customization
4. **Infrastructure** - CI/CD, observability, security
5. **Context** - Domain knowledge & integrations

### Core Philosophy

- **Batteries Included**: Everything needed to start and scale
- **Opinionated Defaults**: Best practices pre-configured
- **Progressive Disclosure**: Simple projects stay simple
- **Knowledge Persistence**: Cross-project learning
- **AI-Native**: Built for human-AI collaboration
